My Projects

I've worked on a lot of cool projects over the past few years. Read about the process below.

Community Fridge Resources

Community Fridge Resources is a web application created to consolidate community fridge resources into one place. This application is the result of the iterative design process, collaboration with four other students from varying disciplines, and multiple rounds of user testing and feedback. This was my first time participating in al stages of the software development process in a team and throughout the semester I learned a lot about how to employ tech for social good as well as how to collaborate virtually and across disciplines.

Learn more about the project by reading about the development process here, by watching a demo video, or by viewing the code.

Ballet Basics

Ballet Basics is an interactive web application that teaches a user the five fundamental ballet positions and provides a quiz for them to test their knowledge. This project was my individual capstone project for User Interface Design in Spring 2020 and allowed me to gain a better understanding of the iterative design process.

Learn more about the project by watching a demo video or viewing the source code.

College Finder

College Finder is a web application that allows users to search college based on attributes including location type (i.e., urban, suburban, or rural), school size, or extracurriculars offered. It also allows user to see similar schools and includes a text search. This project was completed with another student for a virtual course in fall 2020 and I completed it with a greater understanding of postgreSQL databases.
